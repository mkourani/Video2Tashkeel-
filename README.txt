✨ تَشْكِيلٌ - Video2Tashkeel
================================

برنامج تحويل الفيديو إلى نص عربي مشكول
Video to Arabic Text with Tashkeel

📦 الملفات المرفقة:
------------------
app.py              : البرنامج الرئيسي
requirements.txt    : المتطلبات
install.bat         : مثبت تلقائي
run.bat             : مشغل تلقائي
ffmpeg/             : مجلد FFmpeg (أضف ملفات FFmpeg هنا)
output/             : النتائج المحفوظة
temp/               : الملفات المؤقتة

🚀 طريقة التشغيل:
----------------
1. تأكد من تثبيت Python 3.10
2. ضع ملفات FFmpeg في مجلد ffmpeg/
3. شغل install.bat (مرة واحدة فقط)
4. شغل run.bat لتشغيل البرنامج

🔑 مفتاح API:
------------
- افتح: https://aistudio.google.com/apikey
- سجل الدخول بحساب Google
- أنشئ مفتاح API جديد
- استخدم المفتاح في البرنامج

🌐 اللغات المدعومة:
------------------
- الفارسية (Farsi)
- العربية (Arabic)
- الإنجليزية (English)

📝 الملاحظات:
------------
- احفظ مفتاح API في أول استخدام
- اختر نوع المصدر (فيديو/صوت/نص/SRT)
- اتبع المراحل الأربع
- يتم حفظ النتائج تلقائياً في مجلد output/
